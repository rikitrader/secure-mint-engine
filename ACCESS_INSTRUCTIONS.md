# Access Instructions

## Source Code Access

The source code for SecureMint Engine is protected to ensure responsible use and proper licensing.

### How to Get Access

1. **Email Request**: Send an email requesting access
2. **Include in your email**:
   - Your name or organization
   - Intended use case (commercial/educational/personal)
   - Brief description of your project
   - Agreement to the MIT License terms

3. **Wait for Response**: You will receive the access credentials within 24-48 hours
